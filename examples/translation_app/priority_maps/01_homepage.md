# 01_homepage Priority Map

## 页面目标
作为产品核心主页面，让用户快速输入文本或 URL，并进入文档、图片、课堂等翻译场景。

## P0
- input_area：承载用户最核心的输入动作，是页面第一视觉重心。
- bottom_tabs：页面级主导航，当前状态需要明确首页选中。

## P1
- lang_switch：直接影响翻译方向，是核心任务的必要控制。
- upload_action：输入区内的主要扩展入口。
- voice_action：输入区内的主要扩展入口。

## P2
- shortcut_bar：提供文档翻译、图片翻译、课堂翻译三个次要场景入口。
- library_button：进入翻译库，重要但非首页主路径。
- nav_bar：承载品牌与入口，辅助用户确认位置。

## P3
- brand_title：品牌识别，不承担主操作。
- placeholder：输入提示，用户开始输入后消失。
- background_surfaces：页面背景、安全区、弱分隔和辅助占位。
