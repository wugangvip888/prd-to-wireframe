# Intent Rules

## 固定字段

Intent 文件必须包含以下字段，且顺序固定：

```text
页面目标：xxx
用户进来第一件事：xxx
视觉重心：xxx
主要操作：xxx
次要目标：xxx
关键状态：xxx
PRD硬约束：xxx
用户心智：xxx
```

## 写作规则

```text
Intent 只描述页面意图和设计判断，不展开具体布局细节。
页面目标必须服务 PRD 中的核心业务任务。
用户进来第一件事必须描述用户进入页面后的首要动作。
视觉重心必须能指导后续元素权重排序。
主要操作用于识别 P0/P1。
次要目标用于识别 P2/P3。
关键状态只记录影响线框图表达的页面级状态。
PRD硬约束必须来自 PRD 原文，不自行扩展业务规则。
用户心智用于约束页面气质和信息组织方向。
语言应简短、明确，偏判断，不写长段说明。
```

## 强约束触发词

```text
必须、仅、默认、右侧、配套、图标、箭头、固定、隐藏、显示、不展示、无冗余、同步、独立。
```
