> 版本：v1.0

> 状态：正式稿

---

## 一、文档概述

### 1.1 功能定位

**AI创作**是独立的底部 Tab 入口，面向 C 端内容创作者与普通用户，提供两大 AI 视频创作能力：

| 能力 | 定义 |
|------|------|
| AI漫剧生成 | 用户输入剧情梗概/角色设定，一键生成幻灯片式或剧情视频 |
| 爆款视频复刻 | 用户输入爆款视频链接或上传视频，AI 拆解爆款结构并复刻生成 |

---

### 1.2 核心价值主张

- **极简输入**：一句话描述或链接粘贴即可启动创作，无需专业剪辑能力
- **灵感引导**：内置横滑灵感示例卡片，降低空白输入门槛
- **一体化创作**：输入、示例、参数、生成全部聚合在单一主卡内，无多层跳转
- **状态清晰**：文件上传全程卡片化展示，上传中/成功状态一目了然

---

### 1.3 目标用户

| 用户类型 | 核心诉求 | 使用频率 |
|---------|---------|---------|
| 内容创作者 | 快速产出可发布的漫剧或复刻爆款涨粉 | 高频日更 |
| 普通用户 | 把脑洞变成可分享视频 | 低频尝鲜 |
| 营销/运营人员 | 批量生成剧情化内容/复刻行业爆款 | 项目制 |

---

## 二、业务流程总览

### 2.1 主链路

```
进入 AI创作 Tab
    ↓
选择模式（AI漫剧生成 / 爆款视频复刻）
    ↓
    ├─ AI漫剧生成
    │       ↓
    │   选择子模式（幻灯片视频 / 剧情视频）
    │       ↓
    │   输入内容（或点击灵感示例自动填充）
    │       ↓
    │   （可选）上传补充剧本文档
    │       ↓
    │   配置常用参数（画风/时长/比例/配音/字幕）
    │       ↓
    │   点击"立即生成漫剧"
    │       ↓
    │   前端校验 → 内容安全审核 → 额度校验
    │       ↓
    │   ★ 剧情大纲页（确认/编辑标题、概述、人物、场景、分镜大纲）
    │       ↓
    │   ★ 角色设定页（浏览效果图 → 可选进入详情修改形象/音色）
    │       ↓
    │   ★ 场景设定页（浏览效果图 → 可选进入详情修改场景图）
    │       ↓
    │   ★ 漫剧生成中（后台渲染，可后台等待系统通知）
    │       ↓
    │   ★ 漫剧预览与分镜编辑（播放成片 → 按需局部重绘/重新配音）
    │       ↓
    │   导出弹窗（分享给好友 / 保存到手机相册）
    │
    └─ 爆款视频复刻
            ↓
        粘贴视频链接 / 输入复刻方向 / 上传本地视频
            ↓
        （可选）点击灵感示例自动填充
            ↓
        配置常用参数（复刻强度/输出风格/比例/配音）
            ↓
        点击"开始复刻生成"
            ↓
        前端校验 → 内容安全审核 → 额度校验
            ↓
        进入生成队列（后台生成，可后台等待系统通知）
            ↓
        生成预览
            ↓
        导出弹窗（分享给好友 / 保存到手机相册）
```

---

### 2.2 核心设计原则

| 原则 | 说明 |
|------|------|
| 单主卡聚合 | 模式切换、输入框、示例、参数、生成按钮全部在一个主卡内完成 |
| 输入优先 | 输入框为页面视觉中心，所有辅助信息围绕输入框布局 |
| 示例前置引导 | 灵感示例卡片置于输入框下方，帮助用户快速理解产品能力 |
| 状态不打断流程 | 上传中/上传成功以文件卡片形式内嵌在输入框内，不阻断操作路径 |
| 参数轻量化 | 常用参数直接展示，减少决策成本 |

#### 3.1.2 功能描述

AI漫剧生成页面是用户进行漫剧类内容创作的主操作界面。用户通过输入剧情描述（或选择灵感示例填充），选择视频子模式，配置常用参数后点击主 CTA 启动生成任务。

- 视频子模式（幻灯片视频/剧情视频）内嵌在输入框顶部区域，与文本输入形成连续的创作语境
- 灵感示例卡片以单行横滑形式位于输入框下方，降低冷启动门槛
- 支持上传补充剧本文档，以文件卡片形式置于输入框顶部展示全程状态

   - 前端跳转至"生成中状态页"，开始轮询任务进度

5. **后台生成服务**
   - 根据视频子模式（`slideshow_mode` / `story_mode`）、画风、参数调用对应模型管道
   - 如有剧本文档附件，合并到文本上下文后调用生成
